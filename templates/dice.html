Dice Roller

Type a string of the following format, with or without spaces:

x dy ? z
where:
  • x is the optional number of dice to be rolled, if none is given 1 of the specified dice is rolled.
  • dy is the type of die to be rolled, where y is the number of faces on the die.
  • ? is an optional operator for addition (+); subtraction (-); multiplication (*, x or X) or division (/ or รท), used to apply a modifier to the roll.
  • z is an optional modifier, which is applied to the roll with a given operator.
Press one of the following buttons to roll the string in a particular way:
  • Standard Roll the string as standard and return the total.
  • Critical Roll the dice of the string twice then return the total with modifier applied.
  • Advantage Roll the string twice and return both listed highest to lowest.
  • Disadvantage Roll the string twice and return both listed lowest to highest.
  • Abilities Roll 4d6 - the lowest 6 times and return an array ordered highest to lowest.
