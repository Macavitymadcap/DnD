Dangeons & Dragons

Welcome to the home page adventurer!

This is a webapp filled with tools for running 5th edition Dungeons & Dragons.

It has been made with css, html, javascript and python, all hosted as a flask web app.

Try out the following links to aid you in your quest:

  • Dice Roller - Roll a given string in multiple ways.
  • Search Monsters - Type a monster name into the following input and click to launch it's Statblock.
